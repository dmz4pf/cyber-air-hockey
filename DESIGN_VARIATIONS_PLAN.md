# Air Hockey - 5 Unique Frontend Design Variations

## Overview

Each design is a **complete visual overhaul** - not just color themes, but entirely different:
- Layout structures and component arrangements
- Visual styles and rendering approaches
- Typography and iconography
- Animations and effects
- Sound design concepts
- User experience patterns

---

## Design 1: Neon Arcade Cabinet

### Concept
Full arcade cabinet experience - like playing in a 1980s arcade with CRT monitors, coin slots, and cabinet art.

### Visual Elements
- **Frame**: Arcade cabinet border around the game (wood grain texture, metal trim)
- **CRT Effect**: Scanlines, screen curvature, phosphor glow
- **Neon Colors**: Hot pink, electric cyan, yellow accents
- **Typography**: "Press Start 2P" style blocky fonts
- **Coin Counter**: "INSERT COIN" / "CREDITS: 3"

## Implementation Strategy

### Phase 1: Core Design System
1. Create design token system for each variant
2. Build base component variants
3. Implement canvas renderers for each style

### Phase 2: Design-Specific Components
1. Unique layouts for each design
2. Custom animations and effects
3. Theme-specific overlays

### Phase 3: Polish & Integration
1. Design switcher in settings
2. Smooth transitions between designs
3. Persist user preference
