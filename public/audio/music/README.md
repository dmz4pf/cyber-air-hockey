# Music Tracks

Download free synthwave tracks and place them here.

## Required Files

| Filename | Purpose | Duration | BPM |
|----------|---------|----------|-----|
| `menu-theme.mp3` | Menu/lobby music | 2-3 min loop | ~98 |
| `gameplay-loop.mp3` | Active gameplay | 2-3 min loop | ~128 |
| `overtime.mp3` | Overtime/intense moments | 1-2 min loop | ~145 |
| `victory.mp3` | Victory sting | 8-15 sec | ~130 |
| `defeat.mp3` | Defeat sting | 6-10 sec | ~88 |

## How to Add

1. Download tracks from sources above
2. Rename to match filenames in table
3. Place MP3 files in this directory
4. Music will automatically play when available

## Notes

- Music is optional - SFX and ambient work without files
- Tracks should be MP3 format for best compatibility
- Loops should be seamless (no gap at loop point)
- All Pixabay/Mixkit tracks are free for commercial use
