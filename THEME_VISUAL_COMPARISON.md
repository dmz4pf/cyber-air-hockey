## Visual Quality Checklist

For each theme, verify:

- [ ] **Colors**
  - Primary color shows correctly
  - Secondary color visible in gradients
  - Text has proper contrast
  - Borders are visible but not overwhelming

- [ ] **Effects**
  - Glow intensity matches design (8-28)
  - Scanlines overlay (if applicable)
  - CRT curve (if applicable)
  - Trails on moving elements (if applicable)

- [ ] **Typography**
  - Heading font loads and displays
  - Body font is readable
  - Score font is clear
  - Font sizes scale properly

- [ ] **Layout**
  - Components align properly
  - Spacing is consistent
  - Responsive breakpoints work
  - No overflow issues

- [ ] **Animations**
  - Entrance animations smooth
  - Hover effects responsive
  - Page transitions work
  - No jank or lag

- [ ] **Light Theme (Arctic Frost)**
  - Background is light
  - Text is dark and readable
  - Buttons have dark text
  - Sufficient contrast everywhere

## Performance Considerations

### Glow-Heavy Themes (Tokyo, Molten, Synthwave)
- May impact FPS on low-end devices
- Consider reducing glow for mobile
- Use CSS `will-change` for animated elements

### Scanline Themes (Synthwave)
- Fixed background-image overlay
- Minimal performance impact
- Disable on low-end devices if needed

### Light Theme (Arctic)
- Better battery life on OLED displays
- Higher readability in bright environments
- Lower eye strain for extended play

## Accessibility Notes

### All Themes
- Meet WCAG AA contrast requirements
- Keyboard navigable
- Screen reader friendly
- Focus indicators visible

### Special Considerations
- Arctic Frost: Best for visually impaired (high contrast)
- Midnight Club: Good for light-sensitive users (subtle effects)
- Tokyo/Molten: May be intense for some users (provide option to reduce effects)
